## Local Usage

### Prerequisites

- Install the project following the [Installation Guide](../how-to/installation.md).

### Steps

1. Start the local Panel server:
	```bash
	uvx holoviz-mcp-serve
	```
2. Open the URL printed in the terminal. This starts the bundled Panel apps.
